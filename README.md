# Mouse_AVN_single_cell_model
Single cell model of a mouse atrio-ventricular node cell. 

Original publication: 
Bartolucci C, Mesirca P, Ricci E, Sales-Bellés C, Torre E, Louradour J, Mangoni ME, Severi S. Computational modelling of mouse atrio ventricular node action potential and automaticity. J Physiol. 2024 Oct;602(19):4821-4847. doi: 10.1113/JP285950. Epub 2024 Sep 13. PMID: 39269369.

The folder includes:

- a main file (main.m) to launch single-cell simulations
- the model file (AVNmodel_new.m)
- steady-staete initial condition vector in CTRL (CIss_lastMDP_300s.mat) and following 100% funny current block (20241121_lastMDP_pacemaking_0If.mat)
